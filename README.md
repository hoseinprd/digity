<h1 align="center" style="border-bottom: none;">digity</h1>
<h3 align="center">Generate an n-digit code</h3>


## Installation

```sh
$ npm install digity
```

## Usage


```js
// ESM
import { digit } from 'digity';

// CommonJS
const { digity } = require('digity');

const token = digity(6) // 6-digit random number

```

That's all.
